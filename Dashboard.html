🔗 เชื่อมต่อ Google Sheet

วิธี Export Google Sheet เป็น CSV URL:

1. เปิด Google Sheet ของคุณ
2. คลิก File → Share → Publish to web
3. เลือก Sheet และเปลี่ยนเป็น Comma-separated values (.csv)
4. กด Publish แล้ว Copy link ที่ได้
5. วาง URL ด้านล่างแล้วกด "โหลดข้อมูล"
📋 Column ที่รองรับ (ชื่อ Header ใน Row 1):
date หรือ วันที่ · product หรือ สินค้า
quantity หรือ จำนวน · price หรือ ราคา
total หรือ รวม · status หรือ สถานะ
